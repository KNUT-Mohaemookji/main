프로젝트 개요

매끼마다 무슨 요리를 할지 고민하는 자취생들을 위해 유튜브의 다양한 요리

2. System Archtecture

3. Operational Environment


Web Server

:클라이언트의 요청을 받아 정척 웹페이지 반환(메인 페이지)

Crawling Server

:유튜브에서 크롤링한 영상의 소스코드를 카테도리별로 데이터베이스에 저장

Web Application Server

:데이터베이스에 저장된 유튜브 소스코드 영상을 Select하여 동적 웹페이지 반환(요리추천 페이지)

Database

:간편식, 면요리, 밥요리 등 카테고리 별로 유튜브 영상 소스코드 저장

4. Physical Scope


웹 페이지

: HTML/CSS/JavaScript

서버

- 인프라 : AWS EC2
- Web Server & WAS : Node.JS
- Crawling Server : Python/Django
- DBMS : MongoDB
- Container : Docker

프로젝트

- 버전관리 : Git Hub
- CI/CD : Git Actions
- 협업도구 : MicrosoftTeams / Azure Boards
